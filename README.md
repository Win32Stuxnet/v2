# Network Quick Scan made in C# for Windows
---
## Features -
- Allows you to scan a subnet of a network for its hosts, as with IPs
- Easy User Interface made in mind of new networking engineers as with cybersecurity for red teaming. 
**PLANNED**
- Script abilities to make or iuse default premade scripts to scan for network vulns



## Instructions on build 
We believe open source as literal as possible so this will incliude ALL of the cfode for each class, 

- 1. Install .dotnet SDK 
- 2. Go to root project
- 3. Run dotnet build
- 4. run dotnet run
